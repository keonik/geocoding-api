Simple to Integrate

Get started with just a few lines of code

# Get ZIP code information
curl -H "Authorization: Bearer your-api-key" \
"https://geocode-api.com/api/v1/geocode/10001"
# Calculate distance between ZIP codes
curl -H "Authorization: Bearer your-api-key" \
"https://geocode-api.com/api/v1/distance/10001/10002"
# Search ZIP codes by city
curl -H "Authorization: Bearer your-api-key" \
"https://geocode-api.com/api/v1/search?city=New+York&state=NY"
